# UniFontMaker
A font maker for Unitale/CYF (and Minecraft resource pack). WIP.  
Unitale/CYF 字体制作器  

`master` 分支是给 Unitale/CYF 字体用的，可以编译使用  
`remake` 分支目前在重写

`CharacterTemplate` 文件夹下面是常用的字符模板
